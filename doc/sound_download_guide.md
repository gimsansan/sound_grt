# 🎵 오디오 샘플 다운로드 가이드 (자동화 및 수동 방법)

이 문서는 이전에 앱에 적용된 75개의 고품질 어쿠스틱 음원 파일(OGG/MP3)이 어떻게 수집되었는지 그 원리를 설명하고, 앞으로 사용자님이 직접 다른 악기를 찾아서 수동으로 다운로드하는 방법을 안내합니다.

---

## 1. 이전에 음원을 다운로드한 방법 (자동화 스크립트)

제가 백그라운드에서 실행했던 방법은 **GitHub에 공개된 거대한 오픈소스 사운드 폰트 저장소(Repository)에 직접 접근하여 파일을 긁어오는 자동화 스크립트**였습니다.

### 📌 사용한 저장소 (Source)
*   **저장소 이름:** `gleitz/midi-js-soundfonts`
*   **특징:** General MIDI(GM) 표준에 정의된 128가지 악기의 모든 건반 음(88키)을 누군가 최고급 퀄리티로 렌더링해서 `.mp3`와 `.ogg`로 분리하여 웹에 전부 올려둔 방대한 아카이브입니다.
*   **파일 구조 규칙:**
    *   `[악기이름]-[포맷]/[음표].ogg`
    *   예: `acoustic_bass-ogg/C3.ogg`

### 💻 스크립트 원리
1.  앱에서 필요한 **25개 음계 목록**을 배열로 만듭니다. (예: `['C3', 'Db3', 'D3', ... 'C5']`)
2.  Windows 터미널의 `curl` 명령어를 사용하여 반복문(for loop)을 돌립니다.
3.  서버 주소(`https://raw.githubusercontent.com/...`)에 위 규칙을 조합하여 25번 요청을 보내고, 우리 프로젝트의 `assets/sounds` 폴더에 다운로드 후 자동 저장하게 만들었습니다.

---

## 2. 🙋‍♂️ 직접 수동으로 다운로드하는 방법

앞으로 타악기(마림바)나 관악기(플루트) 등 완전히 새로운 악기를 직접 고르고 싶으시다면 아래의 방법을 따라 진행하시면 됩니다.

### Step 1. 사운드 폰트 웹사이트 접속하기
브라우저를 열고 아래 링크 중 하나를 클릭하여 접속합니다.

*   👉 **[전체 악기 목록 보기 (GitHub 웹사이트)](https://github.com/gleitz/midi-js-soundfonts/tree/gh-pages/FluidR3_GM)**

### Step 2. 원하는 악기 폴더 찾기
목록을 보면 엄청나게 많은 폴더가 있습니다. 안드로이드 최적화를 위해 이름 끝에 `-ogg`가 붙은 폴더를 찾으세요.
*   *추천 악기 예시:*
    *   **마림바:** `marimba-ogg`
    *   **바이올린:** `violin-ogg`
    *   **플루트:** `flute-ogg`
    *   **드럼/퍼커션:** `synth_drum-ogg` (또는 타악기 전용)

### Step 3. 음원 파일(Note) 다운로드
원하는 악기 폴더(예: `marimba-ogg`)에 들어가면, `A0.ogg`부터 `C8.ogg`까지 파일들이 쫙 나열되어 있습니다.

1.  원하는 음(예: `C4.ogg`)을 클릭합니다.
2.  화면 우측 상단에 있는 **⬇️ Download (Download raw file)** 버튼을 누르면 즉시 파일이 다운로드됩니다.
3.  다운로드된 파일을 우리 프로젝트의 `assets/sounds/` 폴더로 복사합니다.

> [!TIP]
> **검은 건반(반음) 파일 이름 읽는 법**
> 이 사이트는 샵(#) 대신 **플랫(b)** 표기법을 사용합니다.
> - C# (도 샵) ➡️ `Db` (레 플랫)으로 표기됨 (예: `Db3.ogg`)
> - F# (파 샵) ➡️ `Gb` (솔 플랫)으로 표기됨 (예: `Gb3.ogg`)

### Step 4. 프로젝트에 연결하기
파일을 다운로드했다면, 앱 코드에 연결해야 소리가 납니다.
1. 파일을 `d:\Projects\sound_grt\assets\sounds\` 폴더에 넣습니다. (이름을 `marimba_c4.ogg`처럼 알아보기 쉽게 바꾸셔도 됩니다.)
2. `constants/instruments.ts` 파일을 열어 해당 경로를 `require('../assets/sounds/marimba_c4.ogg')`와 같이 매핑해 주시면 끝납니다!
