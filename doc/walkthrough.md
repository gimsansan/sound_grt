# 대역별 맞춤 3종 현악기(Plucked Strings) 시스템 구축 완료

청능 훈련과 음향학적 분리도에 가장 이상적인 조합인 **'베이스 - 어쿠스틱 기타 - 우쿨렐레'** 시스템으로 성공적으로 전환되었습니다.

## 🛠 주요 구현 내용

### 1. 옥타브 자동 보정 (Auto Octave Mapping)
가장 핵심적인 변경 사항입니다. 사용자는 동일한 C3 위치의 건반을 누르더라도, 선택한 악기에 따라 전혀 다른 대역의 소리를 듣게 됩니다.
*   **🎸 베이스 (저음역):** C3 건반을 누르면 실제로는 아주 낮고 묵직한 **C1** 대역의 베이스 줄 튕기는 소리가 납니다.
*   **🎸 어쿠스틱 기타 (중음역):** C3 건반을 누르면 실제 통기타의 따뜻한 **C3** 소리가 납니다.
*   **🪕 우쿨렐레 (고음역):** C3 건반을 누르면 우쿨렐레 특유의 높고 찰랑거리는 **C5** 대역의 소리가 납니다.

### 2. 악기 전환 탭 및 테마 UI 구현
*   화면 상단에 악기를 실시간으로 변경할 수 있는 **전환 탭(Tab)**을 구현했습니다.
*   단순히 소리만 바뀌는 것이 아니라, 훈련 시 시각적 인지를 돕기 위해 악기별 고유의 **네온 색상 테마**를 적용했습니다.
    *   베이스: 묵직한 **네온 퍼플(Purple)**
    *   기타: 따뜻한 **네온 오렌지(Orange)**
    *   우쿨렐레: 경쾌한 **네온 민트(Teal)**

### 3. 고품질 발현악기(Plucked) 사운드 적용
잔향이 길게 남아 소리가 뭉개지는 피아노 대신, 줄을 튕길 때 타격감이 확실하여 소리가 바로 끊어지는(어택감이 강한) 75개의 고품질 MP3 샘플을 새로 다운로드하여 연동을 완료했습니다.

## ✅ 확인 방법
현재 켜져 있는 터미널에서 `npm start -c` 명령어로 캐시를 초기화하여 서버를 다시 켜주세요. 새로고침을 하면 상단에 악기 선택 탭이 나타나며, 세 가지 악기를 돌아가면서 연주해 보시길 바랍니다! 주파수 대역의 차이가 극명하게 갈리는 것을 체험하실 수 있습니다.
