# 대역별 주파수 분리(Low-Mid-High) 3종 현악기 구현 계획

사용자님이 제시하신 '음역대 마스킹이 없는 완벽한 분리도' 아이디어는 **청능 훈련 앱의 본질에 가장 완벽하게 부합**합니다! 
저음(베이스), 중음(기타), 고음(우쿨렐레/만돌린)은 주파수가 명확히 분리되고 어택감이 확실하여 주파수 변별력 훈련에 최고의 조합입니다. 기존의 피아노/마림바 계획을 전면 수정하여 이 3가지 발현악기(Plucked Strings) 앙상블로 개편합니다.

## User Review Required

> [!IMPORTANT]
> **음원 소싱 및 옥타브 매핑에 대한 확인 요청**
> 1. **음원 확보:** UIowa MIS에는 우쿨렐레가 없으므로, 제가 3가지 악기(어쿠스틱 베이스, 어쿠스틱 기타, 우쿨렐레)의 고품질 오픈소스 음원(총 75개)을 직접 발굴해서 최적화된 `.ogg`로 세팅해 드려도 괜찮을까요?
> 2. **옥타브 자동 시프트:** 현재 화면의 건반은 C3~C5로 고정되어 있습니다. 
>    - 악기로 '베이스'를 선택하고 C3를 누르면 실제로는 **더 묵직한 C1이나 C2 대역의 소리**가 나게 하고,
>    - '우쿨렐레'를 선택하고 C3를 누르면 **찰랑거리는 고음역인 C4나 C5 대역의 소리**가 나도록 (즉, 건반 위치는 같아도 악기별로 가장 듣기 좋은 주파수 대역이 매핑되도록) 자동 보정을 넣는 것이 어떨까요?

## Proposed Changes

### 1. 주파수 대역별 악기 선택 UI 추가 (`app/(tabs)/sound_grt/index.tsx`)
- 피아노 건반 위쪽 빈 공간에 **[ 🎸 베이스 (저음) | 🎸 어쿠스틱 기타 (중음) | 🪕 우쿨렐레 (고음) ]** 형태의 탭(Tab) 버튼을 추가합니다.
- 선택한 악기에 따라 건반을 누를 때 뿜어져 나오는 네온 불빛 색상을 다르게 적용하여 시각적 직관성을 높입니다.
  - 베이스 (저음): 묵직한 **보라색(Purple)**
  - 어쿠스틱 기타 (중음): 따뜻한 **주황색(Orange)**
  - 우쿨렐레 (고음): 찰랑거리고 밝은 **민트/초록색(Teal/Green)**

### 2. 데이터 구조 리팩토링 (`constants/instruments.ts`)
- 기존 피아노 전용 배열을 폐기하고, 악기 타입별로 사운드 뱅크를 관리하는 구조로 변경합니다.
- `INSTRUMENT_SOUNDS.bass`, `INSTRUMENT_SOUNDS.guitar`, `INSTRUMENT_SOUNDS.ukulele` 객체를 생성하여 각각 25개의 건반과 음원 파일 경로를 1:1로 매핑합니다.

#### [MODIFY] [constants/instruments.ts](file:///d:/Projects/sound_grt/constants/instruments.ts)

### 3. 사운드 에셋 구축 (`assets/sounds/`)
- `assets/sounds/bass_c3.ogg`, `guitar_c3.ogg`, `ukulele_c3.ogg` 등의 네이밍 규칙으로 총 75개의 최적화된 음원 파일을 새로 구성합니다. (기존 피아노 파일은 삭제하거나 폴더를 분리하여 정리합니다.)
